About Color Change Boxes

The Color Change Boxes project is an interactive webpage where four different boxes are displayed. Each box is associated with a specific background color, and when you click on a box, the background color of the entire page changes to the color of the selected box.

The boxes also update their content to indicate that they have been clicked, providing immediate feedback to the user. When the mouse leaves the box, its content reverts to its original number. This project demonstrates how to manipulate the DOM using JavaScript, handle events like `click` and `mouseleave`, and change the page's styling dynamically.

The four boxes in this project are:

  • Box 1: Changes the background color to white and updates the box text to "Box1 Clicked".
  • Box 2: Changes the background color to yellow and updates the box text to "Box2 Clicked".
  • Box 3: Changes the background color to gray and updates the box text to "Box3 Clicked".
  • Box 4: Changes the background color to blue and updates the box text to "Box4 Clicked".

This project provides an excellent introduction to handling user interactions and DOM manipulation with JavaScript.
